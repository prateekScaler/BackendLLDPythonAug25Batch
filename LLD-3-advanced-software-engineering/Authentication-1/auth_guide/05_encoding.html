Three transforms, three different purposes

Letter-Based Examples (ELI5)

üìñ Encoding: Pig Latin

Original:  HELLO
Rule:      Move first letter to end, add "AY"
Encoded:   ELLOHAY

Reverse:   Remove "AY", move last letter to front
Result:    HELLO

‚úÖ Anyone who knows the rule can reverse it!

üîí Encryption: Caesar Cipher (shift by 3)

Original:  H   E   L   L   O
Key:       +3  +3  +3  +3  +3  (shift each letter by 3)
Encrypted: K   H   O   O   R

To decrypt, you MUST know the shift amount (key = 3):
K   H   O   O   R
-3  -3  -3  -3  -3
H   E   L   L   O

‚ùå Without the key, you'd have to try all 25 possibilities!

üëÜ Hashing: Letter Sum

Original:  CAT
Process:   C=3, A=1, T=20  ‚Üí  3+1+20 = 24
Hash:      24

Now what word made 24?
- CAT = 24 ‚úì
- TAC = 24 ‚úì
- ATC = 24 ‚úì
- X = 24 ‚úì

‚ùå IMPOSSIBLE to know which one was the original!
   That's why it's called ONE-WAY.

When to Use What?

üìñ

Use Encoding When...

  • Sending binary data in text format
  • URL needs special characters
  • Email attachments
üîí

Use Encryption When...

  • Sending secrets over network
  • Storing API keys you need to retrieve
  • Protecting files
üëÜ

Use Hashing When...

  • Storing passwords
  • Verifying file integrity
  • Digital signatures

Common Mistakes

‚ùå Using Encoding for Security
